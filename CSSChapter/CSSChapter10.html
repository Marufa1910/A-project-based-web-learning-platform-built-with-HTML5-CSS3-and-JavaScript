Chapter 10: Transitions & Animations

Bringing your UI to life with smooth motion and dynamic effects.

10.1 CSS Transitions

Transitions change property values smoothly over a specific duration, usually triggered by an interaction like :hover.

Interactive Demo (Hover the box)
Hover Me
Shorthand Property
transition: [property] [duration] [timing-function] [delay];

10.2 CSS Animations

Animations move through multiple stages automatically using @keyframes without needing a user trigger.

Live Keyframe View
Code Syntax
@keyframes move { 0% { left: 0px; } 100% { left: 200px; } } .box { animation: move 4s infinite; }

Transitions vs Animations

Feature Transition Animation
Trigger Requires interaction (:hover) Starts automatically
Control Simple A to B states Complex multi-step keyframes
Loops Plays once Can loop infinitely
