# Coding Bird - Interactive Web Learning Platform

**Coding Bird** is a modern, responsive e-learning platform designed to help aspiring developers master the future of tech. The project features structured lessons for HTML5, CSS3, and JavaScript, utilizing a project-based learning approach with real-time visual representations of code.



## 🚀 Live Demo
https://marufa1910.github.io/A-project-based-web-learning-platform-built-with-HTML5-CSS3-and-JavaScript/

## ✨ Key Features
* **Structured Curriculum:** Organized chapters covering HTML, CSS, and JavaScript fundamentals.
* **Interactive Code Previews:** Visual representations of CSS syntax and HTML elements to bridge the gap between code and UI.
* **Responsive Design:** Fully optimized for all screen sizes using Bootstrap 5.
* **Modern UI/UX:** Clean, "glassmorphism" inspired design with smooth transitions and professional typography.
* **Resource Library:** Includes quick links to team details, testimonials, and contact forms.

## 🛠 Tech Stack
* **Language:** HTML5, CSS3, JavaScript (ES6+)
* **Framework:** Bootstrap 5.0
* **Libraries:** * [jQuery](https://jquery.com/) (DOM manipulation)
    * [Animate.css](https://animate.style/) (Smooth entrance animations)
    * [Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/) (Interactive sliders)
* **Icons:** Font Awesome 5, Bootstrap Icons
* **Fonts:** Google Fonts (Heebo, Nunito, Fira Code)

## 📖 How it Works
The platform uses a modular JavaScript-driven approach to render lessons. By utilizing a `createSection` function, the app dynamically generates educational cards that contain:
1.  **Instructional Text:** Explaining the concept.
2.  **Syntax Highlighting:** Showing the code structure.
3.  **Live Result:** A visual box showing exactly how the code renders in a browser.
